About C++

C++ is general-purpose object-oriented programming (OOP) language developed by Bjarne Stroustrup.Originally, C++ was called “C with classes,”

as it had all the properties of the C language with the addition of user-defined data types called “classes.” It was renamed C++ in 1983.

C++ is considered an intermediate-level language, as it includes both high and low-level language features.
